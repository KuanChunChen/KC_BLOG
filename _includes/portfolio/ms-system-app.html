CTMS app

該產品有多個產品線需要完成。
我主要負責多個項目的開發，以下是我實現的功能：
1. 包括CTMS app、CCore SDK、Sature splash app、AWS Lambda串接、SGPCheckupdate app等。
2. CTMS 是一款與 Cashub Agent 配合使用的app：我在此專案專注於實現 UI、網路請求、本地儲存、AIDL溝通...等。
3. CCore SDK是專為第三方使用而設計的SDK：它提供了接口實現供第三方開發人員使用。
4. Sature Splash app 作為 POS 機的啟動應用程序，該app監聽Boot開機訊號，實作做頁面重新定向、API請求...等客製化功能。
5. AWS Lambda + S3：串接aws服務進我們的專案，例如：實現滿足某條件時，會像後端服務器發送對應POS的配置指令。
6. SGP Check Update、Ketting System app，旨在讓客戶利用 POS 機的功能，主要透過CTMS app 做UI的調整或是客製化功能的要求。

上圖呈現的是CTMS app主要的UI，主要是給使用者一個介面去配置相關config與server溝通，
以及提供POS機內的資訊，操控底層API...等。
